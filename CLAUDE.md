# Next Supabase Starter

## Descripción
Is a starter to create landings with Next js & supase

## Para quién
All kind of business

## Arquitectura
- Next.js 16 con App Router
- Tailwind CSS v4 para estilos
- Supabase para base de datos
- shadcn/ui para componentes
- Stripe para pagos
- Resend para emails transaccionales
- TypeScript
- Vercel para despliegue

## Estado actual
- Landing page con formulario de waitlist
- Conectado con Supabase (tabla waitlist)
- Sistema de autenticación preparado
- Integración con Stripe configurada
- UI components con shadcn/ui

## Convenciones
- Idioma: español
- Componentes React funcionales con TypeScript
- Variables de entorno en .env.local (no subir a git)
- Usar "use client" solo cuando sea necesario
- Preferir Server Components por defecto
